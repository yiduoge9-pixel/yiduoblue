Project Statement

About this project

This project explores how short digital messages lose emotional and cultural context.

Users input a message, choose a context, reveal a possible misreading, repair the meaning and receive a final communication receipt.

The score is not scientific. It is a design device that makes invisible communication gaps easier to see.

Portfolio framing: Interactive communication / multilingual misreading / creative coding / user-centred revision.
Portfolio Case Study

The Miscommunication Machine

A creative coding prototype that uses interaction design to make hidden tone, cultural context and digital misreading visible, testable and repairable.

Project Question

How can interactive design make invisible digital misreading visible?

My Role

Concept, UX flow, interface design, HTML/CSS/JS, interaction rules and multilingual writing.

Outcome

A live prototype where users write, misread, repair and download a communication receipt.

Interaction Logic

Message → Context → Intention → Misread → Compare → Repair → Receipt.

Technical System

Dynamic rendering, state management, localStorage, modal accessibility and downloadable output.

Portfolio Value

The project shows creative coding, UX structure, cultural translation and reflective prototyping.

Design Process

  1. Concept: I started from the phrase “I'm fine” and explored how short messages force readers to guess emotional intention.
  2. Interaction: I designed an 8-step journey from message input to context, intention, misreading, repair and receipt.
  3. Multilingual Layer: I added Chinese-English and Chinese-Cantonese paths to show how tone, rhythm and cultural meaning shift across languages.

Prototype Testing Plan

5 bilingual users
3 feedback questions
Receipt final artifact

The next testing stage will ask bilingual users to complete the friend conversation and Chinese-Cantonese paths, then identify which step makes the communication gap most visible.

What I Learned

I learned that creative communication is not only about sending a clear message. It is also about designing situations where meaning can be questioned, repaired and reinterpreted.
